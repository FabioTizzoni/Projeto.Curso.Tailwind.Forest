# 🏷️ Forest Hosting - Hospedagem de Sites

Este projeto simula um site de hospedagem, desenvolvida para praticar e demonstrar habilidades avançadas com **Tailwind CSS**. O site foca em uma experiência de usuário (UX) intuitiva, performance otimizada e um design moderno voltado para o setor de tecnologia.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estrutura semântica e acessível.
- **Tailwind CSS (v4.0):** Estilização de alta performance com classes utilitárias.
- **PostCSS:** Processamento e otimização de estilos.
- **JavaScript (ES6+):** Implementação de comportamentos dinâmicos, como:
  - **Menu Mobile:** Alternância de visibilidade da navegação em dispositivos móveis.
